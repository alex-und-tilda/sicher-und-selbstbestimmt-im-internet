# Prüfbericht – Feedback-Seiten

## Neue Lernlogik

```text
Antwort anklicken
Rückmelde-Seite sehen
Erklärung lesen
bei richtiger Antwort: Weiter
bei falscher Antwort: Nochmal versuchen
bei Unsicherheit: Hilfe-Button
```

## Warum

```text
Das Feedback ist sichtbarer.
Die Erklärung bekommt mehr Raum.
Der Weiter-Button steht direkt unter der Erklärung.
Falsche Antworten werden wertschätzend erklärt.
Lernende können noch einmal antworten.
```
