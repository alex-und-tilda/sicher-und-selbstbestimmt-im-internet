# Änderung: Hover-Farben direkt gesetzt

Problem:

```text
Die Hover-Farbe war bei allen Kacheln Türkis.
Grund: Die CSS-Klassen haben nicht zuverlässig gegriffen.
```

Lösung:

```text
Jede Themen-Kachel bekommt ihre Farbe jetzt direkt als CSS-Variable.
Der Hover-Effekt ist nicht mehr abhängig von der Klassen-Erkennung.
```

Gefundene Themen-IDs:

```text
datenschutz, whatsapp, facebook, instagram, youtube, snapchat, tiktok, hilfe
```
