# Inhaltsaudit – didaktische Überarbeitung

## Ziel

Die Inhalte wurden alltagsnäher, verständlicher und pädagogisch sauberer formuliert.

## Korrigiert

```text
komische oder unlogische Beispiele entfernt
Merksätze alltagsnäher formuliert
Modulstarts klarer gemacht
Übungen konkreter gemacht
Quizfragen als Alltagssituationen formuliert
Hilfelogik klarer getrennt
```

## Didaktische Leitlinien

```text
kurze Sätze
konkrete Alltagssituationen
ein Gedanke pro Satz
keine unnötigen Fachbegriffe
klare Handlungsregel
bei Unsicherheit: Unterstützung holen
keine Angstpädagogik
keine falsche Sicherheit
```
