# Merksatz-Didaktik-Fix

## Befund

```text
Auf Modul-Startseiten stand bereits ein Merksatz.
Das ist didaktisch zu früh.
Außerdem wurde „Merksatz“ optisch doppelt angezeigt.
```

## Korrektur

```text
Merksatz-Boxen auf Startseiten entfernt.
Merksatz-Anzeige vereinheitlicht.
Merksatz-Überschrift wird nur einmal angezeigt.
Merksätze bleiben auf passenden Inhaltsseiten und in Merk-Karten erhalten.
```
