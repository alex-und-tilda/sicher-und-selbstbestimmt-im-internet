Der Button auf den Themenkarten hieß vorher:

```text

```

Das war technisch korrekt, aber für Nutzer*innen nicht gut verständlich.

Jetzt heißt der Button:

```text

```

Die Funktion bleibt gleich: Der Button liest nur den Namen des jeweiligen Themas vor.
