# Änderung: Startseite ohne Balken und Navigation

Auf der Startseite werden ausgeblendet:

```text
Fortschrittsbalken
Zurück-/Weiter-Navigation
```

Sichtbar werden sie erst, wenn ein Thema oder ein Lernmodul geöffnet wird.

Grund:

```text
Auf der Startseite wurde noch kein Lernprozess begonnen.
Ein Fortschrittsbalken und Zurück/Weiter wirken dort missverständlich.
```
