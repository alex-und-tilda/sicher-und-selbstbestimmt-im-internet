# ARASAAC-Piktogramme fest einbinden

Die Lernplattform nutzt 17 ARASAAC-Piktogramme. Aktuell werden sie vom
ARASAAC-Server geladen. Damit sie **fest eingebunden** sind (schneller,
offline nutzbar, datenschutzfreundlicher), lade die 17 Dateien einmal
herunter und speichere sie im Ordner **`assets/lessons/`** – mit genau
dem angegebenen Dateinamen. Fertig. Die Seite findet sie automatisch.

## Ein Piktogramm austauschen

1. Auf https://arasaac.org das bessere Piktogramm suchen (deutsche Suche möglich).
2. Herunterladen und unter dem gleichen Dateinamen in `assets/lessons/` speichern.
