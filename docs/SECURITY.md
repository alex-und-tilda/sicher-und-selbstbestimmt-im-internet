## Sicherheitsproblem melden

Wenn Sie ein Sicherheitsproblem auf dieser Webseite finden, melden Sie es bitte direkt an:

**Frank Runge**  
E-Mail: [deine dienstliche E-Mail-Adresse eintragen]

Bitte beschreiben Sie möglichst genau:

- Was ist passiert?
- Auf welcher Seite ist das Problem?
- Welcher Link oder Button ist betroffen?
- Welches Gerät wurde genutzt?
- Welcher Browser wurde genutzt?

## Reaktion auf Meldungen

Ich prüfe gemeldete Sicherheitsprobleme zeitnah.

Wenn das Problem bestätigt wird, wird es nach Möglichkeit behoben.

Wenn es kein Sicherheitsproblem ist, gebe ich eine kurze Rückmeldung.

## Wichtiger Hinweis

Bitte veröffentlichen Sie Sicherheitsprobleme nicht öffentlich, bevor sie geprüft wurden.

So können Nutzerinnen und Nutzer besser geschützt werden.
