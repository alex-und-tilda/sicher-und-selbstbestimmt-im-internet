# Sicher und selbstbestimmt im Internet

Fertige Lernplattform mit 8 Themen:

1. Datenschutz
2. WhatsApp
3. Facebook
4. Instagram
5. YouTube
6. Snapchat
7. TikTok
8. Hilfe bei Problemen

## Lernlogik

```text
Thema auswählen
Lernweg auswählen
Alltagssituation lesen
Antwort auswählen
Rückmeldung lesen
Weiter oder nochmal versuchen
Merk-Karte ansehen
Quiz machen
```

## Wichtig

```text
Es wird kein Name gespeichert.
Es wird kein Lernstand gespeichert.
Es gibt keinen Login.
Es gibt eine Vorlese-Funktion für den aktuellen Lerninhalt.
```

## Pädagogische Struktur

```text
Startseite: Orientierung
Lernseite: einfache Alltagssituation
Übung: zwei Antworten
Richtige Antwort: Erklärung + Merksatz + Weiter
Falsche Antwort: Erklärung + Nochmal versuchen
Unsicher-Button: Hilfe unabhängig von der Aufgabe
Zusammenfassung: wichtigste Regeln
Merk-Karte: Wiederholung
```

## Hinweis zu Leichter Sprache

Die Inhalte sind in Einfacher Sprache formuliert.
Für offizielle Leichte Sprache ist eine Prüfung durch Menschen aus der Zielgruppe nötig.


## Vorlesen

Die Lernplattform hat jetzt eine Vorlesefunktion.

```text
Vorlesen
Langsam vorlesen
Stopp
```

Die Funktion liest nur den aktuellen Lernbereich vor.

Nicht vorgelesen werden:

```text
Footer
Navigation
Logos
technische Hinweise
```

Beim Seitenwechsel stoppt das Vorlesen automatisch.

## Dezente Profil-Einbindung

Die Startseite zeigt nur einen kleinen Ersteller-Hinweis.

werden aber nicht dominant auf der Lernplattform präsentiert.

Ziel:

- Ersteller bleibt transparent sichtbar.
- Google kann die Profilseiten weiterhin über Sitemap und interne Links finden.


## Profil auf Abschlussseiten

Der Ersteller-Hinweis steht nicht mehr auf der Startseite.
Er erscheint dezent auf Abschluss-/Merkseiten bzw. Ergebnisbereichen.

Ziel:

- Ersteller bleibt transparent.
- Profilseiten bleiben über Links und Sitemap erreichbar.
