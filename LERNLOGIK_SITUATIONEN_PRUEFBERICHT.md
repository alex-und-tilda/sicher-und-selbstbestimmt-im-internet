# Lernlogik-Prüfbericht

## Problem

Einige Seiten hatten zu wenig echten Lerneffekt.

Sie bestanden vor allem aus:

```text
Regel
Beispiele
Merksatz
```

Das ist für die Zielgruppe zu wenig handlungsorientiert.

## Neue Lernlogik

Viele Seiten wurden jetzt so aufgebaut:

```text
1. Stell dir vor: Alltagssituation
2. Was ist daran schwierig?
3. Was ist eine sichere Entscheidung?
4. Merksatz
5. Kleine Übung
```
