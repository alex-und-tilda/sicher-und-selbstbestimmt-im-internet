# Änderung am Fortschrittsbalken

Auf der Startseite steht im Fortschrittsbereich jetzt:

```text
Themenübersicht
Wähle ein Thema
```

Vorher stand rechts:

```text
Start
```

Grund:

```text
„Start“ war missverständlich.
Es wirkte wie ein Lernschritt.
Auf der Startseite soll klar sein:
Die Person soll erst ein Thema auswählen.
```
