# Änderung: Merksatz-Layout

Problem:

```text
Merksätze wurden wortweise untereinander angezeigt.
```

Ursache:

```text
Eine alte CSS-Regel hat dem Merksatz-Text zu wenig Breite gegeben
oder den Text falsch umbrochen.
```

Korrektur:

```text
Merksatz-Box volle Breite
Merksatz-Text normaler Satzumbruch
keine Wort-für-Wort-Darstellung
```
